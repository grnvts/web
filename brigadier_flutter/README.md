# Flutter приложение для бригадира (Android)

Мобильное приложение для Android, предназначенное для бригадиров.

## Запуск

### 1. Убедитесь, что бэкенд запущен:
```bash
cd backend
mvn spring-boot:run
```

Бэкенд должен быть доступен на `http://172.20.10.2:8501`

### 2. Установите зависимости:
```bash
cd brigadier_flutter
flutter pub get
```

### 3. Подключите Android устройство:

1. Включите режим разработчика на телефоне:
   - Настройки → О телефоне → нажмите 7 раз на "Номер сборки"
   - Настройки → Для разработчиков → включите "Отладка по USB"

2. Подключите телефон по USB к компьютеру

3. Разрешите отладку на телефоне (появится запрос)

4. Проверьте подключение:
```bash
flutter devices
```

Должно появиться ваше Android устройство.

### 4. Запустите приложение:
```bash
flutter run
```

Или укажите конкретное устройство:
```bash
flutter run -d <device-id>
```

## Важно:

1. **Убедитесь, что телефон и компьютер в одной Wi-Fi сети**
2. **Убедитесь, что бэкенд запущен** на `http://172.20.10.2:8501`
3. **Откройте порт 8501 в файрволе Windows** (запустите PowerShell от имени администратора):
   ```bash
   netsh advfirewall firewall add rule name="Flutter Backend Port 8501" dir=in action=allow protocol=TCP localport=8501
   ```

## Изменение IP адреса

Если IP адрес компьютера изменился:

1. Узнайте новый IP:
   ```bash
   ipconfig | findstr /i "IPv4"
   ```

2. Обновите в файлах:
   - `lib/services/auth_service.dart`
   - `lib/services/order_service.dart`
   - `lib/services/brigade_service.dart`

   Замените `http://172.20.10.2:8501/api` на ваш текущий IP.

## Альтернатива - Web версия

Если нужно протестировать на Web:
```bash
flutter run -d chrome
```

Но для Web лучше использовать `admin_flutter`, который поддерживает и админа, и бригадира.
