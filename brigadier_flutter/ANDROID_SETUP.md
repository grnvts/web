# Настройка для Android устройства

## Шаги для подключения телефона:

### 1. Включите режим разработчика на Android:
- Откройте **Настройки** → **О телефоне**
- Найдите **Номер сборки** и нажмите на него **7 раз**
- Появится сообщение "Вы стали разработчиком!"

### 2. Включите отладку по USB:
- Вернитесь в **Настройки** → **Для разработчиков** (или **Система** → **Для разработчиков**)
- Включите **Отладка по USB**

### 3. Подключите телефон:
- Подключите телефон к компьютеру через USB кабель
- На телефоне появится запрос "Разрешить отладку по USB?" → нажмите **Разрешить**
- Поставьте галочку "Всегда разрешать с этого компьютера"

### 4. Проверьте подключение:
```bash
cd brigadier_flutter
flutter devices
```

Должно появиться ваше Android устройство в списке.

### 5. Запустите приложение:
```bash
flutter run
```

Или укажите конкретное устройство:
```bash
flutter run -d <device-id>
```

## Важно:

1. **Убедитесь, что телефон и компьютер в одной Wi-Fi сети**
2. **Убедитесь, что бэкенд запущен** на порту 8501
3. **Проверьте IP адрес компьютера** - он может измениться! Выполните:
   ```bash
   ipconfig | findstr /i "IPv4"
   ```
4. **Обновите IP адрес в сервисах** если он изменился:
   - `lib/services/auth_service.dart`
   - `lib/services/order_service.dart`
   - `lib/services/brigade_service.dart`
   
   Замените `http://10.102.222.32:8501/api` на ваш текущий IP адрес.

5. **Откройте порт 8501 в файрволе Windows** (нужны права администратора):
   ```bash
   netsh advfirewall firewall add rule name="Flutter Backend Port 8501" dir=in action=allow protocol=TCP localport=8501
   ```

## Проверка доступности бэкенда:

Откройте в браузере на телефоне (в той же Wi-Fi сети):
```
http://10.102.222.32:8501/api/login
```

Должна быть ошибка (не "недоступен"), значит бэкенд доступен.

## Если устройство не видно:

1. Установите драйверы для вашего телефона (обычно устанавливаются автоматически)
2. Попробуйте другой USB кабель
3. Попробуйте другой USB порт
4. Перезапустите ADB: `adb kill-server && adb start-server`
5. Проверьте, что на телефоне включена отладка по USB

## Альтернатива - Wi-Fi отладка:

Если USB не работает, можно использовать Wi-Fi отладку:

1. Подключите телефон по USB один раз
2. Выполните: `adb tcpip 5555`
3. Отключите USB
4. Узнайте IP телефона (Настройки → О телефоне → Статус → IP-адрес)
5. Выполните: `adb connect <IP-телефона>:5555`
6. Теперь можно запускать: `flutter run`
