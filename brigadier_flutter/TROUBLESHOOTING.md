# Решение проблем с подключением

## Проблема: ERR_CONNECTION_TIMED_OUT

### 1. Проверьте IP адрес компьютера

IP адрес может меняться при переподключении к Wi-Fi. Выполните:
```bash
ipconfig | findstr /i "IPv4"
```

Обновите IP адрес в файлах:
- `lib/services/auth_service.dart`
- `lib/services/order_service.dart`
- `lib/services/brigade_service.dart`

### 2. Откройте порт 8501 в файрволе Windows

**Запустите PowerShell или CMD от имени администратора** и выполните:
```bash
netsh advfirewall firewall add rule name="Flutter Backend Port 8501" dir=in action=allow protocol=TCP localport=8501
```

Проверьте, что правило создано:
```bash
netsh advfirewall firewall show rule name="Flutter Backend Port 8501"
```

### 3. Убедитесь, что бэкенд запущен

Проверьте, что сервер слушает на порту 8501:
```bash
netstat -an | findstr :8501
```

Должно быть:
```
TCP    0.0.0.0:8501           0.0.0.0:0              LISTENING
```

### 4. Проверьте доступность с компьютера

Откройте в браузере на компьютере:
```
http://localhost:8501/api/login
```

Должна быть ошибка (не "недоступен").

### 5. Проверьте доступность с телефона

**Убедитесь, что телефон и компьютер в одной Wi-Fi сети!**

Откройте в браузере на телефоне:
```
http://172.20.10.2:8501/api/login
```

(Замените 172.20.10.2 на ваш текущий IP адрес)

### 6. Альтернативное решение - статический IP

Если IP адрес постоянно меняется, настройте статический IP:

1. Откройте **Настройки сети** → **Wi-Fi** → ваша сеть → **Свойства**
2. Выберите **Редактировать** для IP-настройки
3. Выберите **Вручную**
4. Установите:
   - IP адрес: например, `192.168.1.100` (зависит от вашей сети)
   - Маска подсети: `255.255.255.0`
   - Шлюз: IP вашего роутера (обычно `192.168.1.1` или `192.168.0.1`)
   - DNS: `8.8.8.8` и `8.8.4.4`

### 7. Проверка сети

Убедитесь, что телефон и компьютер в одной подсети:
- Компьютер: `172.20.10.2` → подсеть `172.20.10.x`
- Телефон должен быть в той же подсети `172.20.10.x`

Проверьте IP телефона:
- Настройки → Wi-Fi → ваша сеть → Дополнительно → IP-адрес

### 8. Если ничего не помогает

Попробуйте использовать мобильную точку доступа:
1. Включите мобильную точку доступа на телефоне
2. Подключите компьютер к этой точке доступа
3. Узнайте IP компьютера в этой сети
4. Обновите URL в приложении
